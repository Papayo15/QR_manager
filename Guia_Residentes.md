# Guía Simple para RESIDENTES

## ResidenteApp - Android & iOS

---

## 1. INSTALAR LA APP

### ANDROID:
1. Te llegará `ResidenteApp.apk` por WhatsApp
2. Toca el archivo y luego "Instalar"
3. Si pide permiso, acepta
4. Abre la app

### iOS (iPhone):
1. Descarga "TestFlight" del App Store
2. Toca el enlace que te enviaron
3. Toca "Instalar" en TestFlight
4. Abre ResidenteApp

---

## 2. ENTRAR A LA APP

**NO hay registro. Solo necesitas:**

1. Abre ResidenteApp

2. Escribe 2 datos:
   - **Número de Casa**: Tu número (1-100) o "Administración"
   - **Nombre del Condominio**: Unica, Bocamar, Zebrina, etc.

3. Toca "Entrar"

**Ejemplo:**
```
Número de Casa: 25
Nombre del Condominio: Unica
```

**Eso es todo.** La app guardará estos datos y la próxima vez entrarás directo.

---

## 3. CREAR UN CÓDIGO PARA TU VISITA

1. Toca el botón "Generar Código" o "+"

2. Llena solo 2 datos:
   - **Nombre del Visitante**: Juan Pérez
   - **Nombre del Residente**: Tu nombre (María González)

3. Toca "Generar"

4. Te aparecerá un código de 6 letras/números: **ABC123**

**Ejemplo:**
```
Nombre del Visitante: Juan Pérez
Nombre del Residente: María González

Código generado: ABC123
```

---

## 4. ENVIAR EL CÓDIGO

1. El código aparece en pantalla: **ABC123**

2. Cópialo y envíalo por WhatsApp:

```
Hola! Tu código para entrar es:

ABC123

Cuando llegues, dile este código al guardia.
```

**IMPORTANTE:**
- El código tiene **6 caracteres** (letras y números)
- Envía el código de texto, no solo la imagen del QR
- El código no expira por tiempo, es de un solo uso

---

## 5. QUÉ HACE TU VISITA

Cuando tu visita llega a la entrada:

1. **Guardia pregunta**: "¿Código?"
2. **Tu visita dice**: "ABC123"
3. **Guardia escribe** las 6 letras/números en su app
4. **Si es válido**:
   - Pantalla verde
   - Muestra: Casa, Visitante, Residente
   - Puede pasar
5. **Si no funciona**:
   - Pantalla roja
   - "Código inválido o ya usado"

---

## 6. VER TUS CÓDIGOS ANTERIORES

- Toca "Historial" o "Mis Códigos"
- Verás todos los códigos que has generado
- Puedes ver si ya fueron usados

---

## CONSEJOS

### Para residentes:

- El código tiene exactamente **6 caracteres**
- Dile el código completo a tu visita
- El código se usa **una sola vez**
- Si necesitas que entre varias veces, genera varios códigos

---

## PROBLEMAS COMUNES

### "El código no funciona"

1. Ve a "Historial"
2. Verifica que el código no haya sido usado
3. Genera un nuevo código
4. Envíaselo de nuevo a tu visita

### "Mi visita perdió el código"

1. Abre la app
2. Ve a "Historial"
3. Busca el código que generaste
4. Envíaselo otra vez

O simplemente:
- Genera un nuevo código
- Envíaselo por WhatsApp/llamada

---

## RESUMEN RÁPIDO

1. Entra con: Casa + Condominio
2. Genera código con: Visitante + Residente
3. Te da código de 6 caracteres: **ABC123**
4. Envíalo a tu visita por WhatsApp
5. Tu visita lo dice al guardia
6. Listo

---

**Lo más importante:**
- ResidenteApp genera códigos de **6 caracteres**
- Cada código se usa **una sola vez**
- Sin registro, sin contraseñas, solo casa y condominio
